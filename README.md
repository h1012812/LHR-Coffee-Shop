# ☕ LHR Coffee Shop

A modern and cozy coffee shop website built to offer a smooth browsing experience for customers and a simple management workflow for admins.

## ✨ Features
- Browse coffee products through a clean and friendly interface  
- Fully responsive design for all devices  
- Product details with images and pricing  
- Admin dashboard to add, edit, and delete products  
- Order management system  
- Warm and inviting visual theme

## 🗂️ Backend & Database
Built using **PHP** and **MySQL** to handle products, orders, and admin operations.  
All data is stored securely to keep the system organized and reliable.

## 🎨 Branding
The visual identity and layout were designed to reflect a calm, welcoming coffee‑shop atmosphere.

## 💛 About
LHR Coffee Shop was created to deliver a simple, elegant, and functional experience — both for customers exploring the menu and admins managing the shop.
